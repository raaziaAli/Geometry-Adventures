# Angle Adventures

A single-page interactive learning platform for middle school students exploring Grade 7 Unit 7 — *Angles, Triangles, and Prisms* — built with a Jo Boaler growth-mindset approach and Peter Liljedahl's Building Thinking Classrooms framing.

## What's inside

Four sections matching the unit structure:

- **Section A — Angle Relationships** (special angles, adjacent, non-adjacent, solving for unknowns)
- **Section B — Drawing Polygons with Given Conditions** (triangle inequality, unique triangles)
- **Section C — Solid Geometry** (slicing solids, volume, surface area)
- **Section D — Let's Put It to Work** (applied problem solving)

Each section includes thinking tasks, tiered hints, interactive SVG tools (draggable protractor, pattern block assembler, angle labs, classifiers, equation checkers), and self-check lists.

## How to publish on GitHub Pages

1. Create a new repository on GitHub.
2. Upload `index.html` to the repo root.
3. Go to **Settings → Pages** in your repo.
4. Under *Source*, select the branch (usually `main`) and the root folder (`/`).
5. Click **Save**. Your site will be live at `https://<your-username>.github.io/<repo-name>/` within a minute or two.

No build step required — this is a single self-contained HTML file.
